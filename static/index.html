Sandler Call Flow
1. Pattern Interrupt — Disarming opener, ask permission
2. Up-Front Contract — Set 15-min frame, give prospect control
3. Pain Discovery — Open-ended Qs, reflect back, go deeper
4. Negative Reverse — Gently challenge if they downplay
5. Qualify — Urgency (1-10), budget, decision makers
6. Soft Close — "Would 15 min make sense?" Propose day/time
7. Post-Sell — Lock meeting, ask for stakeholders, thank
